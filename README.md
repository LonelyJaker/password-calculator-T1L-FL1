# 密码计算器

根据SOP时间和当前时间自动计算8155加密区动态密码、8255加密区动态密码、工程模式动态密码。

## 功能

- 支持自定义SOP时间（日历选择/手动输入）
- 自动计算2种加密区动态密码
- 每小时自动更新密码
- 适配移动端/PC端

## 使用方法

1. 克隆仓库：`git clone https://github.com/LonelyJaker/password-calculator-T1L-FL1.git`
2. 打开 `index.html` 文件即可使用

## 计算规则

-工程模式动态密码 = (用户设置的SOP时间 × 当前时间(MMDDHH) - 当前小时数) 的最后六位数字
-8155加密区动态密码 = (固定SOP时间20240910 × 当前时间(MMDDHH)) 的最后六位数字
-8255加密区动态密码 = (固定SOP时间20250830 × 当前时间(MMDDHH)) 的最后六位数字
-系统会自动更新当前时间，并每小时重新计算一次密码
-您可以修改SOP时间，重新计算工程模式动态密码
